# ES6
## Map
```
let map = new Map()
map.set(function(){},"vaule")
map.set("size",{"width":10,"height":20})
```

## WeakMap
key값에는 객체만
```
let wmap = new WeakMap()
wmap.set(function(){},"value")
wmap.set({"width":10,"height":20},{"size":"XL"})
```
## Set
key값의 중복이 없음  
중복해서 넣어도 마지막 값만 저장
```
let set = new Set()
set.add(function(){})
set.add(function(){})
set.add({"width":1010,"height":209})
```
